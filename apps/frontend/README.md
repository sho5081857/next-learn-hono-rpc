# Getting started

## Install Node.js

Make sure you have Node.js 18.17 or later installed.

https://nodejs.org/en

## Install the project's packages
```bash
pnpm i
```

## Environment Variables
```bash
cp .env.example .env
```

## Start the development server
```bash
pnpm dev
```

## Generate the production build
```bash
pnpm build
```

## Start the production server
```bash
pnpm start
```
